# Restaurant Le Solstice

Modern website for Restaurant Le Solstice - Smash Burger & Apéros in Saint-Sauveur, Quebec.

## Features

- Modern, responsive design optimized for mobile and desktop
- Bold typography with Bebas Neue and Inter fonts
- Smooth animations and transitions
- Interactive menu showcase
- Photo gallery
- Contact information and map integration
- Social media links (Facebook & Instagram)

## Tech Stack

- Vite
- Vanilla JavaScript
- CSS3 with modern features
- Google Fonts

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## GitHub Pages Deployment

This project is configured for automatic deployment to GitHub Pages:

1. Push to the `main` branch
2. GitHub Actions will automatically build and deploy
3. Site will be available at: `https://[username].github.io/restaurant-le-solstice/`
